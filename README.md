<div class="course-card">
    <h2 style="margin-top:0">📖 课程简介（Course Introduction）</h2>
    <p>本课程是面向国际生的基础课程，旨在让学生用中文掌握计算机领域常见的术语和表达。我们将从 <strong>基本词汇</strong> 学习出发，逐步深入学习<strong>人工智能、大数据</strong> 等前沿话题和求职中必备的中文技能。</p>
  <div class="warning-box">
	<h3 style="margin-top:0; color: #d35400;">⚠️ 先修要求 (Prerequisites)</h3>
  <ul>
      <li>具备基本的<strong>中文对话</strong>能力（Basic Chinese conversation ability）。</li>
      <li>具备基本的<strong>计算机理论知识</strong>（Basic computer science knowledge）。</li>
  </ul>
</div>
